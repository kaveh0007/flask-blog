## Description
This is a project I am currently working on.
In this MVP version I aim to create a responsive blog application where users (authenticated ones) can register/login and can read/write blogs and can comment on other's blogs.

## Technologies Used
- **Frontend**: HTML + Bootstrap + Vanilla JS (Later to be replaced by ReactJS)
- **Backend**: Flask
- **Forms**: WT-Forms
